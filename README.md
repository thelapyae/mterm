# mterm (Burmese Terminal)

A modern terminal emulator with proper Burmese script support, built with Electron and xterm.js.

## Features
- **Burmese Support**: Correct rendering of complex Burmese script using the Padauk font.
- **Interactive Input**: Supports typing and deleting Burmese characters with correct cluster handling.
- **Theme**: Catppuccin Mocha (High-contrast, eye-friendly).
- **Fast Rendering**: Uses the Canvas renderer for smooth performance.
